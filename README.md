# 21DaysSQLChallenge
21 days SQL Challenge

A hands-on journey to strengthen SQL skills with daily practice.  
- **Duration**: 21 days  
- **Focus**: From fundamentals to advanced (queries, joins, subqueries, window functions, CTEs, optimization)  
- **Dataset**: Hospital beds management  
- **Goal**: Build strong SQL skills for data analysis
- **Mid Mini Project**: The Great Pizza Analytics
- **Final Data Analysis Project** : Capstone Project
